VOLVER ARRIBA
Etiqueta Explicacion Ejemplo
HEAD Se coloca al principio del documento,
contiene información sobre la página, y
los enlaces hacia hojas de estilo/scripts.		 <head><title></title></head>
BODY Esta es la etiqueta de cuerpo, en
ella encontraremos toda o la mayoria
de la informacion a mostrar en el navegador.		 <body><title></title></body>
TABLE Genera una tabla; a esta le deberemos añadir
los elementos <th><th> para cada columna
que queramos añadir; <tr><tr> para cada
fila; luego, cada campo con <td><td>		 <table>
  <tr>
    <th>columna1</th>
  </tr>
  <tr>
    <td>dato1</td>
  </tr>
  <tr>
    <td>dato2</td>
  </tr> </table>
P Esta etiqueta representa un párrafo de texto.
Deja siempre un espacio antes y después de cualquier
elemento.		 <p>[texto]</p>
UL
[Unorganized List]		 Declara una lista sin ordenar (es decir,
separada y listada por 'bullet points'. Se
agregan o quitan puntos mediante etiq.
tipo <li>.		 <ul>
   <li>Elemento-1</li>
   <li>Elemento-2</li>
</ul>
OL
[Organized List]		 Declara una lista ordenada/enumerada.
Al igual que en la anterior, esta se compone
de elementos <li></li>		 <ol>
   <li>Elemento-1</li>
   <li>Elemento-2</li>
</ol>
SCRIPT Representa un guión/serie de programas bajo
un lenguaje web (generalmente, JavaScript). Lo
que coloquemos entre estas etiquetas se 		<script></script>
DIV Se emplea para dividir contenidos en partes,
generalmente con el fin de aplicarles estilos/
organizar estos en partes de la página.		 <div>Contenido1</div>
IMG Este elemento muestra por pantalla una imágen
cuyo enlace se dispone con el atributo 'src'.
Se puede omitir/no usa etiqueta de cierre		 <img src="http../imagen1.png"/>
H1...H5 Cabeceras. Muestran texto de manera más vistosa
(H1), y a menor número disminuye su tamaño y énfasis
(H2, H3, H4, H5)		 <h1>Titulo pagina</h1>
<h3>Subtítulo mas pequeño</h3>
<h5>Subtítulo MUY pequeño</h5>
TEXTAREA Muestra por pantalla una zona de texto, editable
y redimensionable (se puede evitar estas dos mediante
el atributo readonly)		 <textarea readonly>
textotextotexto,
textotexto,
textotextotexto
</textarea>
IFRAME Coloca en una página, una ventana (frame) que
accede a otro sitio web/documento, que le especifiquemos
[mediante etiqueta ]		 <iframe></iframe>
NAV Representa el 'menú principal' de navegación
en nuestra página. [Enlaces a diferentes partes
de la misma]		 <nav></nav>
HEADER La información en la parte 'superior' de la
página		 <header>...</header>
FOOTER Pie de la página.
Normalmente en este se muestran copyright,
enlaces importantes...		 <footer>Copyright...</footer>
INPUT Permite al usuario introducir datos.
Tiene diversas funciones, segun el atributo
type que elijamos (text; email;tel;number)...		 <input type="text"/>
BUTTON Es un botón. Como el anterior, tiene
diversas funciones segun el type, y
se utiliza sobre todo en formularios.		 <button type="clear">
Texto del boton
</button>
FIELDSET Genera un cuadro con borde alrededor,
en el que podemos introducir elementos
y cuyo <legend> le pone etiqueta.		 <fieldset>
<legend>Titulo campo</legend>
</fieldset>
STYLE Referencia a una hoja de estilos 'interna'
que posee las reglas CSS disponibles para los elementos
que le indiquemos.		 <style></style>
LINK Se usa, junto a los atributos rel y
href para especificar tanto hojas de estilos
como scripts/guiónes 'externos' al html.		 <link rel="stylesheet"
href="estilos.css"></link>
SELECT Crea un menú desplegable, en el que
colocamos cada opción con etiquetas
<option></option>, atributo interno
value que especifica la opción escogida.		 <select>
<option>opt1</option>
<option>opt2</option>
</select>
